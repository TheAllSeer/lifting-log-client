# calendar swiper

allowing the user to look at previous dates, or even future dates. 

at the top of the screen, on all tabs, should be a left and right arrow (that are non intrusive to the user experience), that allow the user to look at last week or next week. 

clicking on the middle, should allow the user to select any week they want to look at (for ease of access). 

this should be accessible from both Home and the Workouts tabs, but not from the Presets tab.
